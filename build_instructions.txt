## üêõ Troubleshooting

### "Arcade API authentication failed"

**Solution**: Check that your Arcade API key is correct:

```bash
arcade whoami  # Should show your account
```

### "Rate limit exceeded" on carrier APIs

**Solution**: Implement caching (see Cross-Cutting Concerns skill):

```python
@lru_cache(maxsize=1000)
def get_tracking(tracking_num: str):
    # Cache results for 30 minutes
```

### "LLM extracted invalid tracking number"

**Solution**: Add validation before API calls:

```python
if not validate_tracking_format(tracking_num, carrier):
    return "Could you provide the tracking number again?"
```

### "Power BI dataset update failed"

**Solution**: Check Power BI workspace permissions and dataset schema.

### "OAuth token expired"

**Solution**: Arcade handles token refresh automatically. If issues persist, re-authorize in Arcade dashboard.
