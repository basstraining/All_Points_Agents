### Why This Demo?

### What It Does

```python
1. Monitors YOUR Outlook inbox
2. Categorizes YOUR emails (tracking request, complex issue, etc.)
3. Drafts real responses in YOUR Outlook
4. Sends YOU Slack alerts for urgent items
```

### Build Instructions

**1. Create Replit Project**
```bash

# Name: "AllPoints-EmailTriage-Demo"
```

**2. Install Dependencies**
```bash
pip install arcade-ai anthropic google-adk streamlit
```

**3. Connect Arcade**
```python
# In Arcade dashboard:
# 1. Create account at arcade.dev
# 2. Enable Microsoft Graph integration
# 3. Connect YOUR Outlook account
# 4. Get API key
```

**4. Build the Agent**
```python
from arcade_mcp_server import MCPApp, Context
from arcade_mcp_server.auth import MicrosoftGraph
from google.adk.agents import LlmAgent

# Arcade MCP server
app = MCPApp(name="email_triage_demo")

@app.tool(requires_auth=MicrosoftGraph(scopes=["Mail.Read", "Mail.Send"]))
async def triage_inbox(context: Context):
    """Triage YOUR real inbox"""
    
    # Get YOUR unread emails
    emails = await context.outlook.get_unread_emails(max_count=10)
    
    results = {"processed": 0, "auto_resolved": 0, "escalated": 0}
    
    for email in emails:
        # Categorize with LLM
        if "tracking" in email.body.lower():
            # Draft response in YOUR Outlook
            await context.outlook.create_draft(
                reply_to=email.id,
                body=f"Hi, your tracking info is... (demo)"
            )
            results["auto_resolved"] += 1
        else:
            # Tag for review
            await context.outlook.tag_email(email.id, "Needs Review")
            results["escalated"] += 1
        
        results["processed"] += 1
    
    return results

# Google ADK agent
triage_agent = LlmAgent(
    model='gemini-2.0-flash',
    name='email_triage',
    tools=[triage_inbox]
)
```

**5. Create Streamlit UI**
```python
import streamlit as st

st.title("All Points Email Triage - LIVE DEMO")
st.warning("‚ö†Ô∏è This will use YOUR real Outlook inbox!")

if st.button("üöÄ Run Demo (Use My Real Inbox)"):

        result = triage_agent.run("Triage my inbox")
        
        st.success("‚úÖ Demo Complete!")

```
